# 🍳 ChefAI - Asistente Culinario Inteligente

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Code style](https://img.shields.io/badge/code%20style-black-black)](https://github.com/psf/black)

Sistema de IA que recomienda recetas basadas en ingredientes disponibles, utilizando **TF-IDF**, **similitud de coseno** y **redes semánticas** para entender relaciones entre ingredientes.

## ✨ Características

- 🎯 **Recomendación inteligente**: Basada en similitud de ingredientes
- 🔗 **Red semántica**: Entiende sinónimos, sustitutos y complementos
- 🔍 **Filtros avanzados**: Por dieta, tiempo, dificultad, alergias
- 📊 **Explicabilidad**: Muestra por qué recomienda cada receta
- 🚀 **API REST**: Fácil integración con frontend
- 📈 **Rápido**: Respuesta en < 50ms
